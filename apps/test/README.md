# Test App

A React + Vite + TypeScript application that renders a CWL editor.
This app is part of the @theseus-cwl monorepo is intended for internal testing and experimentation within the monorepo.

## Installation

From the root of the monorepo:

```bash
yarn install
```

or if using npm:

```bash
npm install
```

## Running the App

```bash
npm run dev
```

or

```bash
yarn dev
```

This will start the app locally at <http://localhost:5173>.
