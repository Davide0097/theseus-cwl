# Theseus CWL IDE

A React + Vite + TypeScript application that renders a CWL Ide.

## Installation

From the root of the repo:

```bash
yarn install
```

or if using npm:

```bash
npm install
```

## Running the App

From the root of the repo:

```bash
npm run dev
```

or

```bash
yarn dev
```

This will start the app locally at <http://localhost:3002>.
